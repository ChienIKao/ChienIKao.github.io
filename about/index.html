# 高健壹 (Chien-I Kao)

目前就讀於 國立中興大學 資訊工程學系碩士班，同時於 國家高速網路與計算機中心 (NCHC) 擔任 LLM DevOps Intern。

我專注於 高效能運算 (HPC)、大型語言模型 (LLM) 應用 與 演算法優化。從大學時期參與各類程式競賽與國科會計畫開始，我便致力於探索計算效能的極限，並將學術研究轉化為實際的系統應用。

# 研究興趣

  • High-Performance Computing (HPC): 平行運算、效能優化、分散式系統。
  • LLM & DevOps: 大型語言模型部署與維運、自動化流程。
  • Algorithms: 組合優化問題 (如 Nonogram)、電腦對局 (Computer Games)。
